## 一、项目简介

**DowaterCourseExchange **是一个专为国内高校学生设计的换课交流平台，通过简单直观图形化界面操作，任何用户都能够轻松在网站上用**点击课程表的方式**发布自己的换课请求贴，或者用同样的方法搜索符合自己需求换课帖，同时，本项目可使用docker部署图形化管理员Web界面，方便零基础计算机小白为自己的学校建立同样换课交流网站
